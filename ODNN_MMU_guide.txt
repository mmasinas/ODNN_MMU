ODNN Usage Guide

1. Activate conda environment
conda activate ODNN

2. Prepare necessary input data (please see test dataset: /mnt/bigdisk2/HCS_analysis/ODNN_classification/test)

3. Run the script. To see the full help page, type: python /mnt/bigdisk2/HCS_analysis/ODNN_classification/scripts/ODNN_MMU/ODNN.py --help

Please see example usage below:
python /mnt/bigdisk2/HCS_analysis/ODNN_classification/scripts/ODNN_MMU/ODNN.py -i /mnt/bigdisk2/HCS_analysis/ODNN_classification/test/plates_list_test.txt -o /mnt/bigdisk2/HCS_analysis/ODNN_classification/test/ODNN_results -f /mnt/bigdisk2/HCS_analysis/ODNN_classification/test/features_list_test.txt -y /mnt/bigdisk2/HCS_analysis/ODNN_classification/test/training_test_20240208.csv -l Plate,Metadata_WellRow,Metadata_WellColumn -s ORF,Name,Allele,Strain -u Strain

Please note: if the output folder "-o" is not provided, "ODNN_results" will be automatically generated in the directory where the script was executed.
